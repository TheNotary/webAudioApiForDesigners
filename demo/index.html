Play Audio on :hover

We're going to use the Web Audio API here, no Flash, no HTML 5 audio elements. We'll need a respectable browser that isn't trying to kill off the internet (not IE). We'll use a very light weight javascript library to make using the Web Audio API that much easier. 

<script src="javascripts/webAudioApiForDesigners.js"></script>

We're using jQuery in this demo to make selecting things and events easier.

Just Add: One AudioContext
+ One path to a sound file
+ One name for that sound's buffer!

This script simply loads a sound file, and then when you mouse over a sensative area, it sounds off at the precise time of the javascript call!
